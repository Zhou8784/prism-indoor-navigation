# 棱镜 · 智慧校园高精度室内导航系统

## 项目简介
棱镜是一款面向大型教学楼的轻量化室内导航 Web 应用，深度融合课表数据与室内路网，实现“课表一键导航”与精准的跨楼层路径规划。系统完全基于前端技术，无需蓝牙、WiFi 探针等外部硬件，仅需静态服务器即可部署，成本趋近于零。

## 核心优势
- **课表联动导航**：粘贴课程文本自动解析教室号，每节课均可一键从主入口导航至目标教室。
- **高精度路径规划**：自主构建分层走廊拓扑图，通过房间中心点向走廊投影并校验连接点，消除穿墙误差；基于 Dijkstra 算法实现同层及跨楼层最短路径搜索。
- **零硬件依赖**：纯 HTML/CSS/JavaScript 实现，使用 Leaflet 轻量地图库与 LocalStorage 存储数据，无服务端，无第三方定位硬件。
- **2D 倾斜视角**：利用 CSS 3D Transform 实现地图的透视倾斜效果，增强空间感知（非真实 3D 渲染）。
- **多模态交互**：支持地图点选、类别筛选、模糊搜索、房间列表弹窗等多种终点输入；提供可配置的课前提醒，实现“提醒—导航”闭环。

## 技术栈
- **地图引擎**：[Leaflet](https://leafletjs.com/) (Simple CRS 坐标系，自定义多边形矢量渲染)
- **前端**：原生 JavaScript ES6、[Tailwind CSS](https://tailwindcss.com/)（CDN）、[SweetAlert2](https://sweetalert2.github.io/)
- **数据处理**：正则解析课表 / 教室号，LocalStorage 持久化课表
- **算法**：Dijkstra 最短路径、点至线段投影、走廊多边形包含检测

## 数据说明
- 地图数据（`MAP_DATA`）由三栋建筑（1、2、3 栋）的一层与二层真实房间多边形坐标、走廊中心线构成，存储于 `index.html` 的 `<script>` 标签中。
- 课表解析支持“3栋2楼203 多媒体教室”等格式，自动转换为内部 `roomId` 进行路径规划。
- 默认内置一份示例课表，用户可自行导入替换。

## 快速开始
1. 克隆或下载本仓库。
2. 直接用浏览器打开 `index.html`，或部署到任意静态服务器。
3. 点击侧边栏 📤 按钮导入课表，或直接点击课表卡片上的“导航”按钮体验路径规划。

## 主要功能
- ✅ 房间多边形按设施类型着色显示
- ✅ 楼层独立显隐（1F / 2F）
- ✅ 点击房间弹出信息，支持点选起点/终点
- ✅ 课表文本导入并自动解析教室号
- ✅ 课前 5/10/15 分钟提醒（SweetAlert2 弹窗）
- ✅ 跨楼层路径规划（自动匹配最近楼梯间）
- ✅ 分类筛选弹窗导航、模糊搜索定位
- ✅ 导航路线动画跟随与楼层自动切换
- ✅ 双击地图重置视角
- ✅ 移动端响应式布局，底部抽屉式侧边栏

## 注意事项
- 本系统使用 Leaflet Simple CRS 模拟平面坐标系，不支持真实地理坐标。
- “3D 视角”为 CSS 伪 3D 透视效果，功能按钮已保留但仅作用于视觉倾斜。
- 项目为比赛原型，当前仅覆盖 1、2、3 号教学楼的 1-2 层；扩展时只需在 `MAP_DATA` 中按标准格式添加建筑、楼层及房间数据即可。
